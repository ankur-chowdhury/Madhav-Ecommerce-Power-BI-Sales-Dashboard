# Madhav E-Commerce Business Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard developed to analyze sales performance and business insights for an e-commerce dataset. The dashboard transforms raw transactional data into meaningful visual insights to support data-driven decision making.

## Key Features
- KPI metrics for Total Sales, Profit, Quantity Sold, and Average Order Value
- State-wise sales analysis
- Category and sub-category performance insights
- Customer contribution analysis
- Payment mode distribution
- Monthly profit trend analysis
- Interactive slicers for state and quarter filtering

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Visualization
- Data Modeling

## Key Insights
- Maharashtra generated the highest sales revenue.
- Clothing category contributed the largest share of total quantity sold.
- Cash on Delivery is the most frequently used payment mode.
- Certain product sub-categories like printers and bookcases produced the highest profit margins.
